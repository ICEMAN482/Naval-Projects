# ⚓ Naval Engineering & Automation Portfolio

Welcome to my repository for marine engineering, fluid dynamics, and naval automation mini-projects. This portfolio bridges the gap between physical mechanical systems and modern software logic using microcontrollers like the ESP32 and Arduino.

## 📂 Project Directory

### 1. [Naval Automated Bilge & Telemetry Test Rig](./Bilge-System)
A scaled-down, automated fluid-management safety system mirroring real-world ship bilge operations. 
* **Hardware:** ESP32, L293D Motor Driver, ACS712 5A Current Sensor, Float Switches, Submersible Water Pumps.
* **Key Features:** Automated flood evacuation, live engine load telemetry (via active current sensing), physical reserve puddle management (to prevent impeller cavitation), and software state-flag logic for manual override integration.

## 🛠️ Core Hardware & Tech Stack
* **Microcontrollers:** ESP32
* **Sensors:** ACS712-05B Current Sensors, Magnetic Float Switches
* **Actuators:** 5V Mini Submersible Water Pumps, DC Motors, L293D Motor Driver
* **Languages:** C++ (Arduino IDE)
